# RESCUE - REdshifted Star ClUster Exploration
Marta Reina-Campos (November 2025)
---

Main repository of the website for the RESCUE collaboration, and of the scripts to compare an observed distribution of GCs against different galactic components.

See [website](https://mreinacampos.github.io/starclusters-in-jwst/)

See the directory code_poisson_comparison and the [README](https://github.com/mreinacampos/starclusters-in-jwst/blob/main/code_poisson_comparison/README) for more details on the comparison method.
